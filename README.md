# DSA Practice Repository

## Day 1: Java Basics

This folder contains the foundational concepts of Java programming, essential for Data Structures and Algorithms (DSA) practice.

### Topics Covered:
- **Basics.java**: Introduction to Java syntax, including package declaration, class structure, and the main method. Demonstrates a simple "Hello World!" program with detailed Javadoc comments explaining keywords like `package`, `class`, `public`, `static`, `void`, and `String`.
- **DataTypes.java**: Explores all primitive data types in Java (byte, short, int, long, float, double, char, boolean) and reference types (String). Includes examples of variable declarations, initializations, and output, with Javadoc explaining each type's range and usage.
- **InputOutputs.java**: Covers input and output operations using `System.out.println()`, `print()`, and `printf()`. Demonstrates reading input with `Scanner` for various data types, including format specifiers and escape sequences.
- **Loops.java**: Illustrates all loop types in Java: for, while, do-while, and enhanced for-each loops. Each example includes use cases, such as iterating over arrays or conditional repetitions, with Javadoc explanations.
- **Conditions.java**: Demonstrates conditional statements: if, if-else, if-else if-else, and switch-case. Shows decision-making based on conditions, with examples like checking number signs and day-of-week selection.
- **Methods.java**: Introduces method creation, including static methods, void methods, and methods with return values. Examples include greeting, addition, and message retrieval, with Javadoc detailing method components and usage.
- **TimeNSpaceComplexity.java**: Explains time and space complexity concepts in DSA, including Big O notation (O(1), O(log n), O(n), O(n log n), O(n^2), etc.). Provides code examples for each complexity type, such as binary search for O(log n) and nested loops for O(n^2).

### Key Learnings:
- Understanding Java syntax and structure.
- Working with data types, input/output, loops, conditions, and methods.
- Analyzing algorithm efficiency through time and space complexity.

### How to Run:
1. Ensure Java is installed on your system.
2. Compile and run each `.java` file using `javac` and `java` commands, or use an IDE like VS Code.
3. For example: `javac Day_1/Basics.java && java Day_1.Basics`

This Day 1 foundation sets the stage for advanced DSA topics in subsequent days.
